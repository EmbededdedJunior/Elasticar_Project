# IP기반 Capstone Design 졸업작품II

차체 길이 조절 기능이 탑재된 차량

## 프로젝트 소개
지식재산을 기반으로 제작된 학부과정 졸업 작품이며, 소형 리니어 액추에이터(Linear Actuator)를 이용하여 차량 앞,뒤 길이를 조절할 수 있는 기능이 탑재된 무선RC카 입니다.
